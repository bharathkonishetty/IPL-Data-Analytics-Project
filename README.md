# IPL Data Analytics Project

## Overview

This project analyzes IPL (Indian Premier League) ball-by-ball data using Python, Pandas, and Matplotlib to derive insights about player performance, team performance, scoring patterns, and bowling effectiveness.

## Dataset
### Dataset Source
IPL Deliveries Dataset obtained from Kaggle.
* IPL Deliveries Dataset
* 260,000+ ball-by-ball records
* 17 columns including batter, bowler, runs, wickets, innings, and teams

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* VS Code

## Analysis Performed

* Data Cleaning and Missing Value Analysis
* Top 10 Run Scorers
* Top 10 Wicket Takers
* Top 10 Six Hitters
* Team-wise Total Runs
* Strike Rate Analysis
* Economy Rate Analysis
* Batter Runs vs Strike Rate
* Top Batting Partnerships
* Runs by Over Analysis

## Key Insights

* Virat Kohli emerged as the highest run scorer.
* Leading wicket-taking bowlers were identified.
* Strike rate analysis highlighted aggressive batters.
* Economy analysis identified economical bowlers.
* Death overs contributed significantly to scoring.
* Batting partnership analysis revealed successful batting combinations.
